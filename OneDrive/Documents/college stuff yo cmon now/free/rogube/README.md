# Rogube - Binding of Isaac Clone

A 2D top-down roguelike shooter inspired by The Binding of Isaac, built with Phaser 3 and TypeScript.

## Features

- Player cube that moves with WASD or Arrow keys
- Mouse-controlled shooting
- Enemy spawning and AI
- Collision detection
- Bullet physics

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Then open your browser to the URL shown (usually `http://localhost:5173`)

### Build for Production

```bash
npm run build
```

## Controls

- **Movement**: WASD or Arrow Keys
- **Shooting**: Click and hold mouse button (aims at cursor)
